CommandForm

public class CommandForm: NSObject, NSCoding

Form of a command.

This class contains data in order to create Command with ThingIFAPI.postNewCommand(commandForm:completionHandler:).

Mandatory data are followings:

  • Schema name
  • Schema version
  • List of actions

Optional data are followings:

  • Title of a schema
  • Description of a schema
  • Meta data of a schema
